# Puzzle Suite

This project is a browser-based tiling puzzle game built as a static site for GitHub Pages.

## Live Demo

Play it here:

[https://xinyuyua.github.io/puzzle-suite-for-kids/](https://xinyuyua.github.io/puzzle-suite-for-kids/)

## Local Use

Open [index.html](/Users/xinyuyuan/workspace/game1/index.html) in a browser.

## Publish To GitHub Pages

1. Push this folder to a GitHub repository.
2. In GitHub, open `Settings` -> `Pages`.
3. Under `Build and deployment`, choose `Deploy from a branch`.
4. Select branch `main` and folder `/ (root)`.
5. Save, then wait about a minute for the site to publish.

Your game will be available at:

```bash
https://YOUR_USERNAME.github.io/YOUR_REPO/
```
